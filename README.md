# Exercícios de JavaScript

Coleção de exercícios práticos de JavaScript com interfaces HTML, cobrindo formulários, cálculos e interatividade.

## Sobre

Este repositório contém exercícios desenvolvidos durante o aprendizado de JavaScript, com foco em manipulação do DOM, eventos e cálculos dinâmicos em páginas web.

## Tecnologias

- JavaScript
- HTML5
- CSS3

## Exercícios

| Arquivo | Descrição |
|---|---|
| `index.html` | Página de login com validação |
| `desconto.html` | Calculadora de desconto |
| `salario.html` | Cálculo de salário líquido |
| `ir.html` | Cálculo de Imposto de Renda |
| `menu.html` | Menu de navegação interativo |

## Como usar

Abra qualquer arquivo `.html` diretamente no navegador — não é necessária instalação.
